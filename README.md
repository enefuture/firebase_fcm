# firebase_fcm
Google 云消息推送 node的简单服务封装
# 配置安装
你的app需要接入firebase服务，官网：https://firebase.google.com, 并将Firebase Admin SDK添加到项目依赖中，请执行 npm install 安装相关依赖。
注意服务器必须运行 Node.js 4.0+。
